# Start Here

這個 vault 不是單一主題知識庫，而是混合型錄音 archive。

目前內容同時包含：
- 家庭錄音
- 小孩練習與表演
- 演講稿與現場分享
- 醫療 / 學術內容
- 會議與技術討論

所以這套 wiki 的重點不是單一專科整理，而是：

1. 把 raw transcript 穩定轉成 source notes
2. 把內容分流到可重用的 domain pages
3. 把低訊號檔案和真正有價值的內容分開

## 最常用的 5 句指令

```text
請先閱讀 transcripts/AGENTS.md，之後都依照這個 schema 維護我的 handyrecorder wiki。
```

```text
請 ingest 這篇：
transcripts/<檔名>.md
```

```text
這篇 raw transcript 我修過了，請重新 re-ingest 最新版本。
```

```text
請找出最近 1 天更新過的 raw transcripts，重新 re-ingest。
```

```text
請對這個 handyrecorder wiki 做一次 lint。
```

## 建議維護節奏

- 有新錄音轉好就 ingest
- 原文修正後就 re-ingest
- 每累積一批內容就做一次 lint
- 開始真的用 wiki 查內容時，把高價值答案回寫成 `questions/` 或 `patterns/`

## 這個 vault 的起手式

先不要一次整理全部檔案。

先從這 5 類開始長出結構：

1. 家庭對話
2. 演講與簡報
3. 醫療與學術內容
4. 會議與討論
5. 音樂練習與表演
